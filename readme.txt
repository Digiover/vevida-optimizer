Vevida Optimizer: Configurable Auto-Update

== Description ==

This plugin extends the automatic update feature already present in WordPress. The core updates can be switched on or off, 
while the plugin updates can be configured on a per-plugin basis.

== Installation ==

1. Upload the package contents to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How can I configure Auto-Updates? =

The plugin adds a menu item to Dashboard, called Updates Settings. The core updates can be switched on or off, and individual plugin updates can also be configured.
